The best writing advice I know is almost embarrassingly simple: write the way you'd
explain it to a friend across the table.

When you imagine that friend, the throat-clearing disappears. The jargon goes. You
stop trying to sound smart and start trying to be understood — which, it turns out,
is what actually sounds smart.

Read it back out loud. If you'd never say it that way, don't write it that way.
